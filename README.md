# AI-CTF-Agent-Lab

AI-CTF-Agent-Lab is a lightweight experimental project for using AI agents to assist CTF challenge analysis.

The project focuses on structured prompts, Python helper scripts, runtime log analysis, and repeatable workflows for Web, Crypto, Reverse, and Misc challenges.

## Features

- Challenge information extraction
- Web CTF request and response analysis
- Crypto reasoning notes
- Reverse engineering workflow templates
- Python solve script generation prompts
- Runtime error log summarization
- Flag pattern extraction
- Final writeup generation
