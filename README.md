# AI-CTF-Agent-Lab

AI-CTF-Agent-Lab is a lightweight experimental project for using AI agents to assist CTF challenge analysis.

## Features

- Challenge information extraction
- Web CTF request/response analysis
- Crypto formula reasoning notes
- Reverse engineering workflow templates
- Python solve script generation prompts
- Error log summarization
- Final writeup generation

## Workflow

1. Paste challenge description and known files
2. Summarize important clues
3. Generate possible solving paths
4. Create or refine Python scripts
5. Analyze runtime output
6. Produce final writeup

## Example Use Cases

- Web: parameter fuzzing, response comparison, endpoint discovery
- Crypto: RSA, ECDSA, hash collision, CRT fault attack
- Reverse: string extraction, validation logic recovery
- Misc: steganography, audio spectrum, image metadata

## Project Goal

The goal of this project is to reduce repetitive manual work during CTF training by combining structured prompts, AI reasoning, and Python automation scripts.
